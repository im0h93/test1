🔥 إعداد Firebase — 4 خطوات بسيطة

اتبع الخطوات وسيعمل نظامك مع قاعدة بيانات حقيقية مشتركة

1 أنشئ مشروع Firebase

افتح console.firebase.google.com ← أنشئ مشروعاً جديداً ← اختر اسماً مثل hr-system

2 فعّل Firestore و Authentication

من القائمة الجانبية: Firestore Database ← أنشئ قاعدة بيانات ← اختر Test mode مؤقتاً

Authentication ← Get started ← فعّل Email/Password

3 احصل على بيانات الإعداد

Project Settings ← General ← Your apps ← Web app ← Register app ← انسخ الـ firebaseConfig

4 الصق بيانات Firebase هنا
⚠️ ستُحفظ هذه البيانات في المتصفح (localStorage) — لا تشاركها مع أحد
